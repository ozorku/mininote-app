# NoteApp UI

#### Minimal Notes App UI built with React-native

![alt text](./preview.png)

## How to Run App

1. cd to the repo
2. Run Build for either OS

- for iOS
  - run `react-native run-ios`
- for Android
  - run `react-native run-android`
